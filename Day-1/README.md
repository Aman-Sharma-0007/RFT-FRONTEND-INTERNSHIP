# Counter App 🔢

A simple Counter Application built with **React.js** as part of the **GOW AI Academy React.js Internship — Day 1**.

## ✨ Features

- ➕ Increment counter
- ➖ Decrement counter
- 🔄 Reset counter to zero
- 🚫 Prevents negative values (bonus)
- 🔢 Step increment — +1, +5, +10, +20 (bonus)
