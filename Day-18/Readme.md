# 🖥️ Admin Dashboard UI

A responsive Admin Dashboard built with **React.js**, **Vite**, and **React Router DOM** as part of the GOW AI Academy RFT Frontend Internship — Day 18.

---

## 🚀 Tech Stack

- React.js
- Vite
- React Router DOM
- Recharts
- CSS (custom)

---

## ✨ Features

- 📊 **Dashboard** — Stat cards with key metrics and a line chart for user growth
- 👥 **Users Page** — Data table with live search/filter by name or email
- ⚙️ **Settings Page** — Basic account settings form
- 🔀 **Sidebar Navigation** — Active link highlighting with React Router DOM

## 🛠️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/admin-dashboard.git
cd admin-dashboard
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```text
http://localhost:5173
```
