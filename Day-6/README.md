# ✅ TaskMate — Advanced To-Do App

> **GOW AI Academy | RFT Frontend Internship — Day 6 Project**

---

## 📌 Project Overview

An Advanced To-Do App built with **React.js + Vite** as part of the RFT Frontend Internship Day 6 assignment.  
Focuses on managing complex state using state arrays, conditional rendering, and item updates.

---

## 🎯 Concepts Tested

- ✅ State Arrays
- ✅ Updating Items
- ✅ Conditional Rendering
- ✅ Managing Complex State

---

## ⚙️ Features

### Core Tasks
- ➕ Add Task
- 🗑️ Delete Task
- ✔️ Mark as Completed

### Bonus Features
- ✏️ Edit Task
- 🔍 Filter Tasks — All / Completed / Pending

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Aman-Sharma-0007/RFT-INTERNSHIP-FRONTEND/tree/main/Day-6
cd Advanced-to-do
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm run dev
```

### 4. Open in browser
http://localhost:5173
