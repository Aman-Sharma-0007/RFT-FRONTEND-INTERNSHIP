# 💰 Expense Tracker App

A React.js project built during **GOW AI Academy RFT Frontend Internship - Day 20**.

## 🚀 Tech Stack

- React.js
- Vite
- React Router DOM

## ✨ Features

- ➕ Add expenses with title, amount, category, and date
- ✏️ Edit existing expenses
- 🗑️ Delete expenses
- 📊 Filter expenses by category
- 💵 View total expenses
- 📅 History page with category-wise breakdown

## 📁 Project Structure

src/
├── components/
│   ├── Navbar.jsx
│   ├── ExpenseForm.jsx
│   └── ExpenseList.jsx
├── pages/
│   ├── Home.jsx
│   └── History.jsx
├── App.jsx
└── main.jsx

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Aman-Sharma-0007/RFT-FRONTEND-INTERNSHIP/tree/main/Day-20.git

# Navigate to project folder
cd expense-tracker

# Install dependencies
npm install

# Install React Router DOM
npm install react-router-dom

# Start development server
npm run dev

```

## 📸 Pages

- **Home** – Add, edit, delete and filter expenses
- **History** – View all expenses and category-wise totals

- State management with objects
