# 🌙 Theme Toggle App

A React.js project built during the **RFT Frontend Internship (Day 3)**.  
This app demonstrates light/dark mode toggling using React state, Context API, React Router, and localStorage persistence.

---

## 🚀 Tech Stack

- React 18
- Vite
- React Router DOM v6
- Context API
- CSS (no external UI library)

---


## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Aman-Sharma-0007/RFT-INTERNSHIP-FRONTEND/tree/main/Day-3
cd theme-toggle
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

in your browser.

---

## ✨ Features

- 🌙 Toggle between Light and Dark mode
- 💾 Theme preference saved in localStorage
- 🌐 Multiple pages using React Router DOM
- 🔗 Active link highlighting with NavLink
- ⚡ Fast development with Vite

---

## 🧠 Concepts Covered

| Concept | Where Used |
|---|---|
| `useState` | Managing dark/light state |
| `useEffect` | Syncing theme to localStorage |
| Context API | Sharing theme across all pages |
| React Router DOM | Navigation between pages |
| Conditional Styling | Switching CSS classes on state |
| localStorage | Persisting theme on refresh |
